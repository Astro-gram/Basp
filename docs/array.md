# Arrays

### Add To Array

```c
array numbers = [1, 2, 3]
numbers.push(4)

//Result: [1, 2, 3, 4]
```

### Index Array

```c
array numbers = [1, 2, 3]
numbers[0]

//Result: 1
```

### Array Properties & Methods

```c
array numbers = [1, 2, 3]
numbers.length

numbers.push(5)
numbers.pop(0)


//Result: 3
```
